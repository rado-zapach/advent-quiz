Rules:

  • A question opens each day at 10:00.
  • The winner for the day is a person with the first correct answer.
  • Changing your answer is permitted, but it also changes the submission time of your answer.
  • Hiding, moving or changing a question clue is prohibited.
  • First 5 persons with a correct answer for the day are awarded points. Five points for the first one ... one point for the fifth one.
  • The person with the most points wins the main prize.
  • Rules specified in a question take priority over these ones.
